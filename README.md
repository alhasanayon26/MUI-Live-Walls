# MIUI Live Wallpapers Pack

A collection of all **six official MIUI Live Wallpapers**, packaged for easy installation on supported MIUI devices. This package is intended for users who have already installed **HyperUnlocked** and want to enjoy the complete set of premium MIUI live wallpapers.

The wallpapers are provided as individual APK files, making installation quick and straightforward without modifying the system partition.

---

## Features

- 🌄 Includes all **6 official MIUI Live Wallpapers**
- ✨ Original MIUI wallpaper experience
- 📱 Compatible with **MIUI 12, MIUI 13, and MIUI 14**
- 📦 Easy APK-based installation
- ⚡ Lightweight and simple to install
- 🔄 No system partition modifications required

---

## Requirements

- MIUI 12, MIUI 13, or MIUI 14
- **HyperUnlocked** installed (recommended)

> **Compatibility:** Designed for MIUI devices. Functionality on HyperOS has not been fully verified and may vary depending on the device model and software version.

---

## Installation

1. Download the latest release.
2. Extract the downloaded ZIP archive using any file manager.
3. Open the extracted folder.
4. Install all **six APK files** one by one.
5. Restart your device (recommended).
6. Open the Wallpaper settings and enjoy the complete MIUI Live Wallpapers collection.
